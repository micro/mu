FAQ

Is Mu free to use?
Yes! Create an account and start using Mu immediately at no cost.

Can I self-host Mu?
Absolutely. Mu is open source and runs as a single Go binary. Check GitHub for install instructions.

What about pricing?
Mu is free with 10 daily AI queries. Need more? Pay as you go from 1p per query. No subscriptions, no tricks. See our plans for details.

How is this different from big tech platforms?
No ads, no algorithmic feeds, no data mining. Just simple, useful tools that work for you.

Can AI agents use Mu?
Yes. Mu supports the Model Context Protocol (MCP). Connect any MCP-compatible AI client to /mcp using a Bearer token. Agents get the same free tier (10 AI queries/day) and pay-as-you-go rates as human users.
